## ADDED Requirements

### Requirement: Admin settings bar
Админ-панель SHALL отображать компактную верхнюю панель с переключателями активного AI-провайдера и музыкального бэкенда. Панель SHALL быть первой строкой интерфейса админки, перед табами навигации, и отображаться на всех вкладках без дублирования.

#### Scenario: Bar renders at top
- **WHEN** администратор открывает любую вкладку админ-панели
- **THEN** верхняя панель с переключателями провайдера ИИ и источника музыки отображается перед табами навигации

#### Scenario: Switching AI provider
- **WHEN** администратор выбирает другой провайдер ИИ в верхней панели
- **THEN** вызов `POST /api/admin/settings/provider` выполняется с новым ID провайдера
- **THEN** сегментед-контрол обновляет активное состояние без перезагрузки страницы

#### Scenario: Switching music backend
- **WHEN** администратор выбирает другой музыкальный бэкенд в верхней панели
- **THEN** вызов `POST /api/admin/settings/backend` выполняется с новым ID бэкенда
- **THEN** сегментед-контрол обновляет активное состояние без перезагрузки страницы

#### Scenario: Settings not duplicated
- **WHEN** администратор переключается между вкладками админ-панели
- **THEN** переключатели отображаются ровно один раз (в верхней панели)
- **THEN** ни одна вкладка не содержит дублирующихся переключателей внизу
