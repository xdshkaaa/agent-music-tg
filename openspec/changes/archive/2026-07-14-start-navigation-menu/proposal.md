## Why

Сейчас `/start` показывает только одну инлайн-кнопку (WebApp) и текстовый список команд. Пользователю приходится либо печатать команды вручную, либо открывать Mini App. Добавление навигационного меню из инлайн-кнопок сделает бота заметно удобнее — все основные функции будут в одном tap.

## What Changes

- Заменить одиночную WebApp-кнопку в `/start` на меню из 3–4 рядов инлайн-кнопок
- Кнопки: «Открыть приложение» (WebApp, full-width), «Купить» / «Генерация» / «Профиль» / «История» / «Поддержка», и условная «Админка» для админов
- Добавить callback-хендлеры `nav:*` для обработки нажатий
- Вынести тела команд в отдельные функции, чтобы их можно было вызывать и из `bot.command()`, и из `nav:*` колбэков
- Сократить текст сообщения `/start` — убрать дублирующийся список команд
- `/app` оставить без изменений

## Capabilities

### New Capabilities
- `start-navigation`: навигационное инлайн-меню в команде `/start` с кнопками для всех основных разделов бота

### Modified Capabilities

(нет)

## Impact

- `server/bot/index.ts` — изменить хендлер `/start`, добавить callback-хендлеры `nav:*`
- `server/bot/shop.ts` — вынести тело `/buy` и `/profile` в отдельные функции
- `server/bot/generate.ts` — вынести тело `/generate` в отдельную функцию
- `server/bot/history.ts` — вынести тело `/history` в отдельную функцию
